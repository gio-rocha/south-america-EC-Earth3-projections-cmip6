# South_America_EC-Earth3_Projections_CMIP6

# Overview
  This repository contains Python scripts developed to analyze future climate projections over South America using EC-Earth3 CMIP6 model output.

# Objectives
 - Evaluate long-term projected temperature changes over South America based on EC-Earth3 CMIP6 projections
 - Compare present and future climate scenarios
 - Generate spatial maps for SSP2-4.5 and SSP5-8.5 scenarios (2081-2100)

# Data Sources
 - Original CMIP6 data: https://esgf-ui.ceda.ac.uk/cog/search/cmip6-ceda/
 - 1 CMIP6 model output: EC-Earth3

# Tools and Libraries
 - Python
 - Xarray
 - Numpy
 - Matplotlib
 - Cartopy
 - Geopandas

# Methodology
 - Mean temperature change (ºC)
 - Total precipitation change (%)
 - SSP2-4.5 and SSP5-8.5 scenarios
 - Historical period: 1995-2014
 - Future period: 2081-2100
   
# Outputs
 - Spatial maps for South America
 - Continental climate change analysis (temperature and precipitation)
